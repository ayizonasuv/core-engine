## Installation
------------

### Prerequisites

* Docker (version 20.10 or later)
* Docker Compose (version 1.28 or later)
* Node.js (version 16 or later)
* npm (version 7 or later)

### Installation Steps

1. Clone the repository using Git:
   ```bash
git clone https://github.com/core-engine/core-engine.git
```
2. Navigate to the project directory:
   ```bash
cd core-engine
```
3. Create a new file named `.env` in the project root and add the following environment variables:
   ```
DB_HOST=localhost
DB_USER=your_username
DB_PASSWORD=your_password
DB_NAME=your_database_name
```
4. Build and start the Docker containers:
   ```bash
docker-compose up -d
```
5. Install the required dependencies using npm:
   ```bash
npm install
```
6. Start the development server:
   ```bash
npm start
```
